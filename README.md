# Customer_churn_analysis
## Data Processing & Visualization Workflow

1. **Data Cleaning (Python)**
   - Handled missing values and inconsistent data.
   - Converted categorical values into meaningful formats.
   - corrected data types.

2. **Exploratory Data Analysis (EDA)**
   - Performed EDA using Python (Pandas, Matplotlib, Seaborn).
   - Analyzed customer demographics, contract types, tenure distribution, and monthly charges.
   - Identified patterns and factors contributing to customer churn.

3. **Data Visualization (Power BI)**
   - Exported the cleaned dataset for visualization.
   - Built an interactive **Power BI dashboard** to monitor key metrics and churn trends.

4. **Key Metrics in Dashboard**
   - Total Customers
   - Churned Customers
   - Churn Rate
   - Average Monthly Charges
   - Average Tenure
   - Total Revenue
